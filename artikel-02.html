Raspberry Pi mit HiFiBerry DAC+ Standard RCA, Volumio, LCD Anzeige und TL-MR3020

Von Jörg Neikes am 30.07.2016 veröffentlicht.

Der Raspberry Pi als mitnehmbare Soundstation.

Ja, es ist Lego, der Klassiker wenn mal eben was braucht um einen Raspberry Pi zu verkleiden, weil die Gehäse, die man hat wie immer natürlich nicht passen.

Nun aber mal zur benötigten Hardware:

Raspberry Pi B+ oder höher HiFiBerry DAC+ Standard RCA RGB LCD Shield von Adafruit Industries TL-MR3020 von TP-LINK ®

Software:

Volumio

Ich werde hier nur die Änderungen beschreiben, die von der Standard installation abweicht. Eine Englische Version ist in meinem Github Repository zu finden unter: Volumio-LCD

Der Kode für das Adafruit_CharLCDPlate ist von: https://github.com/spotrh/rpihacks/tree/master/Adafruit_CharLCDPlate

Geschriben wurde das von R.U.Sirius und wurde gefixed von aubreykloppers. Für mehr Informationen siehe: https://aubreykloppers.wordpress.com/2014/01/30/raspberry-pi-volumio-a-new-music-player-and-it-works-out-the-box

Das Python script showSongInfo.py ist im Original von http://didtheypush.blogspot.de/2013/10/raspyfi-and-adafruit-16x2-char-lcd-plate.html

Die reparierte Version von https://www.dropbox.com/s/iqoyy5lm2tik1my/LCD%20Python%20Script.zip sollte benutzt werden!

Alle Arbeiten wurden im /home/volumio Ordner auf dem Raspberry Pi ausgeführt!

You have to do the following things listed on https://learn.adafruit.com/adafruit-16x2-character-lcd-plus-keypad-for-raspberry-pi/usage

$
sudo apt-get update
sudo apt-get install python-smbus
sudo apt-get install i2c-tools
sudo apt-get install build-essential python-dev python-smbus python-pip git
sudo pip install RPi.GPIO
git clone https://github.com/adafruit/Adafruit_Python_CharLCD.git
cd Adafruit_Python_CharLCD
sudo python setup.py install

Bei der Frage, ob alle Pakete installiert werden sollen mit "yes" y + Enter antworten!

Folgendes muss noch in die Datei /etc/modules eingetragen werden: i2c-bcm2708 i2c-dev

nano /etc/modules + Enter

i2c-bcm2708
i2c-dev

Nun noch die config.txt in /boot anpassen:

nano /boot/config.txt + Enter

device_tree=bcm2708-rpi-b.dtb
device_tree_param=i2c1=on
device_tree_param=spi=on
dtparam=i2c1=on
dtparam=i2c_arm=on

!!! Neustart !!!

Jetzt erst kann das script ausgeführt werden, da die entsprechenden Module geladen sind!

sudo i2cdetect -y 1 + Enter # Für die neue Raspberry Pi 2 Hardware Revision 2 512MB

or

sudo i2cdetect -y 0 +Enter # Für die alte Raspberry Pi 1 Hardware Revision 1 256MB

Kopiert nun die lcdSongInfo.py Datei ins Adafruit_CharLCDPlate Verzeichnis.

$
cp lcdSongInfo.py /home/volumio/Adafruit_CharLCDPlate/

Damit das Script nicht immer per Hand gestartet werden muss legen wir noch einen Autostart an:

sudo crontab -e + Enter

@reboot sudo python /home/volumio/Adafruit_CharLCDPlate/lcdSongInfo.py

Nun noch das Volumio System konfigurieren und neustarten. Das LCD startet automatisch.
